# homebrew-mapture

Homebrew tap for [Mapture](https://github.com/mandotpro/mapture.dev).

## Install

- Stable formula will appear after the first semver release is published.
- `brew install mandotpro/mapture/mapture-canary` for the rolling canary channel.

Add the tap once:

```bash
brew tap mandotpro/mapture
```

Then install or upgrade as needed:

```bash
brew upgrade mapture-canary
```

This repository is updated by the release automation in [mandotpro/mapture.dev](https://github.com/mandotpro/mapture.dev).
